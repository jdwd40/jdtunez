Mode Bar — 3 approaches

A floating strip that lets the listener switch how they're listening: studio mix, live version, demo, lyrics, radio, etc. Explores how many options fit, and how the current mode is signaled.

A · horizontal tabs + scrub
B · segmented dial
C · stacked with preview
